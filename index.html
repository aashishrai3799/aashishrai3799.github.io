EXPERIENCE


1. Undergraduate Research Intern - Shared Reality Lab, McGill University, Canada

(May/2020 - Oct/2020)
Worked on a research project to improve the existing state-of-the-art algorithm for Semantic Face Editing (control any specific face attribute keeping others unaltered) of generative network outputs in terms of perceived quality of facial features.
Modified the framework to make it equally usable for various other complex attributes like race, face shape, etc.


2. Product Development Intern - Drapezy Pvt Ltd.

(May/2020 - Jun/2020)
Worked with the co-founding team to design, implement and test various object detection and extraction techniques using Deep Learning and Image Processing.
Devised algorithms for automated perspective crop and contour filling (filling up the white-spaces) in cropped images using OpenCV, Python.

3. Undergraduate Researcher - MLCV Lab , ECED, SVNIT
(in collaboration with Norwegian Biometrics Laboratory, NTNU, Norway)

(Dec/2019 – Apr/2020)
Designed a CNN based, light weight, progressive residual propagating asymmetrical architecture with three modules (LF, HF feature extraction and reconstruction) to generate 8x upscaled (Super Resolution) images from 8x8, 16x16, 24x24 LR images.
The model gave appreciable results on benchmark datasets CelebA (PSNR: 26.55) and LFW (PSNR: 26.26).


4. Summer Internship - IIRS, Indian Space Research Organization (ISRO)

(May/2019 – Jul/2019)
Co-guide: Dr. Anil Kumar (Scientist IIRS/ISRO & Head PRSD)
Implemented computationally efficient algorithms for the pixel-wise classification of Panchromatic (single band) and Multispectral (up-to 15 bands) satellite images using ANN and CNN.
Devised algorithms for Hard and Soft fuzzy classification and created an Interactive Graphical User Interface.


5. Undergraduate Researcher - MLCV Lab, ECED, SVNIT

(Jan/2019 – Nov/2019)
Designed an Automated Attendance System using Deep Learning to mark the attendance of entire class simultaneously and overcome usual challenges of occlusion, orientation and luminance(brightness).
Used Transfer Learning to utilize the features of pre trained Inception model on new dataset and achieve adequate results along with making the training computationally feasible.
Designed an Interactive Graphical User Interface for the entire system.
The system could automatically generate subject-wise attendance sheets.
